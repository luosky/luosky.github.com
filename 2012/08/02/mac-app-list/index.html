Mac App 列表

Aug 2nd, 2012 • Permalink

编程

  • Dash : 文档查找 和 代码snippet管理
  • SourceTree : 版本控制

命令行增强:

  • Oh my zsh: 各种强化
  • autojump :  快速进行目录跳转
  • TotalTerminal : 随时切出控制台

笔记

  • Mou : markdown编辑器
  • nvAlt : evernote的替代品,支持markdown,支持wiki式的笔记链接,更适合用来自己写笔记,evernote则适合当做知识库,用来存别人写的东西.
  • Evernote
  • Clearly Evernote的插件,highlight功能非常不错,很适合边看文章边划重点.也是从遥远的年代就希望evernote加上却一直没加上的功能.如果还能加上类似pdf的笔记功能就更完美了.
  • Sublime Text 2

辅助

  • Alfred : spotlight替代品
  • witch : 窗口切换应用,我主要用来切换当前应用的多个窗口
  • slate : 窗口布局管理,支持多显示器.我的配置
  • BetterTouchTool : 触摸板增强, 支持很多手势
  • Afloat : 让某个窗口”总是在前”
  • TRIM Enabler SSD硬盘增强
  • second bar : (在多个显示器里都显示menu bar)
  • Bartender 隐藏/合并menubar上的图标
  • DejaMenu : 随时随地呼出菜单栏  
  • RescueTime : 记录时间
  • The Unarchiver : 解压软件,可避免windows里打包的文件解压乱码.
  • Dropbox
  • FinderPath  : 在finder显示路径
  • DTerm : command + shift + enter 快速执行命令行
  • clipmenu ：http://www.clipmenu.com/
  • accessmenubarapps :    http://www.ortisoft.de/en/accessmenubarapps/

设计:

  • SparkBox : 截图管理
  • Xee : 看图软件
  • paparazzi : 网页截屏

阅读

  • calibre : 电子书格式转换

music:

  • MusicSeekerX 高音质音乐下载
  • soulseek 音乐下载
  • 更新歌词及ID3 tag等:
    • DynamicLyrics (Menubar显示歌词,推荐)
    • Tagalicious 批量更新封面,标题等tag
    • nLyrics : (改ID3 tag 编码, 解决windows下的tag在mac下乱码的问题)
    • gituns
    • get Lyrical
    • TunesArt : 更新mp3的专辑封面
